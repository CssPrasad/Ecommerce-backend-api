# E-Commerce Backend API

This is a basic backend API using Django REST Framework.

## Features
- Product management
- JWT Authentication (coming soon)
- PostgreSQL integration (optional)

## Setup
```bash
pip install -r requirements.txt
python manage.py runserver
```
